Proyecto:
Desarrollar un sistema biométrico de acceso remoto capaz de almacenar datos en un servidor web propio mediante Wi-Fi y, posteriormente, visualizarlos.

  Nombre de la conexión Wi-Fi
  Contraseña de la conexión Wi-Fi

  Ip fija al ESP32
  Puerta de Entrada al Router
